◆ is_collide()

def is_collide (   self)

is_collide function checks the collision between the bullet and enemy.

It returns boolean variable true if the bullet hits the enemy and false of it doesn't This function uses the distance formula to calculate distance between bullet and enemy. distance == square root of ( (x2-x1)^2 + (y2-y1)^2 )
